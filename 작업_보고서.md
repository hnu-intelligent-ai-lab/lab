# 🏥 연구실 홈페이지 구축 및 배포 완료 보고서

오늘 진행된 모든 작업 내역과 최종 배포 과정에 사용된 주요 명령어들을 정리했습니다.

## 1. 주요 수정 사항
- **이미지 추가**: 'Activities' 섹션에 Google AI 대상 수상 이미지 추가 및 크기 최적화 (`max-height: 320px`).
- **내비게이션 개선**: `IntersectionObserver`를 활용한 **Scroll Spy** 기능을 구현하여 사용자의 스크롤 위치에 따라 메뉴가 자동으로 하이라이트되도록 수정.
- **오타 수정**: 연구원(성찬민)의 프로필 사진 경로 및 이메일 주소 오류 수정.
- **코드 정리**: 배포 주소 최적화를 위해 불필요한 `CNAME` 파일 삭제.

## 3. 사용된 주요 Git 명령어 기록
작업 중 사용된 실제 명령어 흐름입니다. 나중에 참고하실 때 활용하세요.

### 저장소 연결 및 초기화
```powershell
# Git 초기화
git init

# 원격 저장소 연결 (단체 계정 저장소로 설정)
git remote set-url origin https://github.com/hnu-intelligent-ai-lab/lab.git
```

### 파일 업로드 및 배포 (CNAME 삭제 포함)
```powershell
# CNAME 파일 삭제 (주소에서 아이디 노출 방지)
git rm CNAME --ignore-unmatch
# 또는 수동 삭제 후 반영
rm CNAME

# 변경 사항 스테이징 및 커밋
git add .
git commit -m "Migration to organization for clean URL"

# 최종 푸시 (main 브랜치)
git push -u origin main
```
